Αρχικά απαιτείται η εγκατάσταση της γλώσσας προγραμματισμού Python και της εφαρμογής DB Browser for SQLite για την διαχείριση της βάσης.
Από τις βιβλιοθήκες της γλώσσας χρειάζονται οι tkinter (για την γραφική διεπαφή) & sqlite3 (για την σύνδεση με τη βάση δεδομένων).
Η εικόνα AEROPLANO_DB.png ,το αρχείο κώδικα airport_code.py και η βάση AERODROMIO_FINAL.DB πρέπει να βρίσκονται στον ίδιο φάκελο.
